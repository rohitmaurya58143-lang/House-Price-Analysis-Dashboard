# 🏠 House Price Analysis Dashboard

## 📌 Overview

This project presents an interactive Power BI dashboard to analyze house prices using a real-world dataset.
It focuses on understanding how different property features and amenities affect pricing.
## ⚙️ Tools Used
* Power BI
* Dataset: Housing.csv
## 📊 Features
* KPI Cards:
  * Total Houses
  * Average Price (in ₹ Lakhs)
  * % Houses with AC
  * % Houses on Main Road

* Interactive Filter:

  * Filter by Furnishing Status

* Visualizations:

  * Price comparison based on property features
  * Price trends across different house configurations
  * Furnishing distribution (Donut Chart)

## 💡 Key Insights

* Property features significantly impact price variation
* Furnished houses are generally more expensive
* Houses located on main roads have higher value
* Amenities like air conditioning increase property price
